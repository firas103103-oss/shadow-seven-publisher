# Shadow-7 Publisher — مواصفات النظام (Excel/CSV)

## الملفات المتوفرة

| الملف | المحتوى |
|-------|---------|
| **01_ARCHITECTURE.csv** | الهيكل العام، الطبقات، المسارات، الحدود |
| **02_PATHS_AND_FILES.csv** | مسارات الملفات والمكونات الرئيسية |
| **03_API_ENDPOINTS.csv** | نقاط نهاية API (طريقة، مسار، وصف) |
| **04_COMPONENTS.csv** | مكونات الواجهة والـ utils |
| **05_TECH_STACK.csv** | التقنيات المستخدمة |
| **06_CONFIG_AND_PORTS.csv** | الإعدادات والمنافذ |
| **07_OMNI_PIPELINE_STAGES.csv** | مراحل خط أنابيب Omni (1-7) |
| **08_DATABASE_SCHEMA.csv** | جداول وقواعد البيانات |

## كيفية الفتح في Excel

1. افتح Excel
2. File → Open
3. اختر الملف `.csv`
4. تأكد من ترميز UTF-8 لدعم العربية

أو: انقر مرتين على الملف لفتحه مباشرة.
